inicializar o projeto
```
sudo yarn init -y
```

adicionar express
```
sudo yarn add express
```

instalar as tipagens da biblioteca
```
sudo yarn add @types/express -D
sudo yarn add typescript -D
```

fazer a tradução 
```
sudo yarn add ts-node-dev -D

-- settings.json

 "scripts": {
    "dev" : "ts-node-dev src/server.ts"
  }

```
